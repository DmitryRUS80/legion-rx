LEGION RX v2.3 — СПОРТИВНЫЙ РЕГЛАМЕНТ

Обновление GitHub Pages:
1. Распакуйте ZIP.
2. В репозитории legion-rx нажмите Add file → Upload files.
3. Перетащите ВСЁ содержимое папки Legion_PWA_v2.3.
4. Нажмите Commit changes.
5. Подождите публикацию GitHub Pages.
6. На iPhone полностью закройте Legion RX и откройте снова.

Если iPhone показывает старую версию:
Настройки → Safari → Дополнительно → Данные сайтов → github.io → удалить,
затем открыть сайт в Safari и снова запустить приложение.

НОВОЕ В v2.3:
- спортивные тай-брейки Best 3;
- жеребьёвка только при абсолютном равенстве;
- DNF с ручным порядком схода;
- DNS и DSQ;
- автоматическая финальная лестница;
- B полностью формирует главный финал A;
- защита от повторяющихся мест;
- исправление сохранённых квалификаций и финалов;
- изменение квалификации сбрасывает зависимые финалы;
- итоговые очки: 25,20,16,13,11,10,8,6,4,3,2,1;
- встроенное краткое описание регламента в Настройках.

LEGION RALLYCROSS MANAGER — ЗАПУСК НА IPHONE

Важно: PWA нельзя установить, если просто открыть index.html как файл.
Проект должен быть опубликован по HTTPS.

БЫСТРЫЙ СПОСОБ ЧЕРЕЗ NETLIFY DROP
1. На компьютере распакуйте ZIP.
2. Откройте страницу https://app.netlify.com/drop
3. Перетащите в окно всю папку Legion_RallyCross_Manager_v1.2_PWA.
4. Netlify выдаст адрес вида https://....netlify.app
5. Откройте этот адрес на iPhone именно в Safari.
6. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх).
7. Выберите «На экран Домой».
8. Нажмите «Добавить».
9. Запускайте приложение с иконки Legion RX.

После первого открытия приложение сохраняет основные файлы и работает офлайн.
Данные гонки хранятся в памяти Safari на этом iPhone.
Не удаляйте данные Safari для сайта, иначе локальное сохранение исчезнет.
Используйте «Экспорт JSON» для резервной копии соревнования.

ОБНОВЛЕНИЕ ПРОГРАММЫ
1. Загрузите новую версию файлов на тот же адрес сайта.
2. Откройте приложение при наличии интернета.
3. Полностью закройте и снова откройте Legion RX.

ВИЗУАЛЬНЫЕ ИЗМЕНЕНИЯ
Да, внешний вид можно менять в style.css: размеры, шрифты, фон, карточки,
цвета, стартовую решётку и мобильную компоновку. Логику гонки это не ломает,
если не менять названия HTML-классов и идентификаторов.
